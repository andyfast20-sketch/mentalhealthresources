# Mental Health Resources Hub

BrightMind is a single-page web experience showcasing curated mental health resources across crisis support, therapy, education, and self-guided tools.

## Running locally
1. Install dependencies (Flask):
   ```bash
   pip install flask
   ```
2. Start the server on http://127.0.0.1:5001:
   ```bash
   python app.py
   ```
3. Open the site in your browser to explore filters, glow mode, and resource cards.

## Features
- Eye-catching glassmorphism design with glow mode toggle.
- Tag filtering to quickly surface urgent help, mindfulness tools, or community spaces.
- Curated crisis lines, therapy directories, education hubs, apps, and safety-planning guides.
